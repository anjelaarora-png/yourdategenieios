## Subtitle (30 chars max) — 3 options, pick one

### Option 1 (recommended)
```

```
**Char count:** 23 / 30 ✓
**Why:** Plants "AI" for category discovery, plants "romance" for emotional hook, mirrors "Romance, Reimagined" tagline.

### Option 2
```

```
**Char count:** 24 / 30 ✓
**Why:** Pulls the hero copy from the date plan card. Lyrical. Brand-perfect.

### Option 3
```
60 seconds to your perfect date
```
**Char count:** 30 / 30 ✓
**Why:** Concrete promise + time-to-value. Strongest conversion frame.

---

## Decision

Default to **Option 1** unless you have a strong pull elsewhere. Subtitle is searchable, so "AI Dates" earns category SEO that "Tonight, something good" doesn't.
